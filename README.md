# 文字列頻出単語検出ツール

## 概要
文章内の単語の出現頻度を解析し、上位20件を表示するツールです。

## 使い方
① テキストを入力  
　・テキストファイルをドラッグ＆ドロップ  
　・または文章を貼り付け  

② 抽出したい文字列を選択
　・高度な解析を選んだ場合右側の単語単位か文章単位を選んでください。

② 除外したい単語を入力（任意）  
　タグ方式で表示されｘクリックで削除できます。 

③ 「解析実行」をクリック  

④ 頻出単語が表示されます（上位20件）  

⑤ 抽出結果から任意のワードをクリックすると除外に加えて再抽出します。
　・追加された除外ワードタグをクリックすると削除できます。

⑥ 必要に応じてCSVでダウンロード可能  

## 使用用途
・長文テキストの傾向をざっくり把握したいとき  
・頻出ワードの確認  
・文章の特徴分析の簡易チェック  

AIを使うほどではない軽い分析用途を想定しています。
## 機能
- 単語頻度カウント
- 除外ワード指定
- 上位20件表示
- CSVダウンロード
- ダークモード切替

## 技術
- HTML / CSS / JavaScript
- 正規表現による文字クレンジング
- 配列操作によるデータ処理

## 工夫点
- メモリ使用量を考慮し最大処理数を制限
- textContentによるXSS対策
- Blob + BOMでCSV出力対応
- CSVインジェクション対策（先頭文字エスケープ）
